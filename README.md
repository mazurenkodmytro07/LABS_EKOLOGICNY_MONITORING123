# Lab 3: Air Quality Monitoring System

## Опис проекту

Цей проект є системою моніторингу якості повітря, що використовує індекс якості повітря (Air Quality Index, AQI) для оцінки рівня забруднення повітря в різних містах. Використовуються п'ять основних параметрів: PM2.5, PM10, NO2, SO2, O3. Система дозволяє здійснювати розрахунок AQI на основі цих показників та виводити їх у вигляді зрозумілої шкали, яка класифікує стан повітря як “добре”, “помірне забруднення”, “шкідливо” або “дуже шкідливо”.

## Проект складається з:

- **Backend**: Node.js, Express, MongoDB.
- **Frontend**: React.js.
- **Бібліотеки**: Chart.js для графіків, Axios для запитів API, Zod для валідації даних.

## Технології

- **Backend**: Node.js, Express, Mongoose, MongoDB
- **Frontend**: React.js
- **Бібліотеки**: 
  - Chart.js для графіків
  - Axios для запитів до API
  - Zod для валідації даних

### Крок 1: Клонування репозиторію

```bash
git clone https://github.com/mazurenkodmytro07/LABS_EKOLOGICNY_MONITORING123.git

Крок 2: Встановлення залежностей

Для кожної лабораторної частини виконайте наступні команди:

Встановлення залежностей для lab3:

Крок 3: Налаштування середовища

У корені репозиторію створіть файл .env і додайте наступні змінні:

Для запуску серверів виконуйте наступні команди:

Для запуску сервера lab1:

Для запуску сервера lab2:
